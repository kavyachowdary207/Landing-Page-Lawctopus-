Lawctopus Landing Page
A modern, responsive, and conversion-focused landing page developed as part of the Lawctopus Frontend Development Assignment. The project redesigns the course landing page with an improved user experience, better visual hierarchy, and responsive layouts while maintaining a professional and accessible design.
